Modelado 3D de superficies

Un modelo de superficie es un vaciado fino que no tiene masa ni volumen; no tienen espesor.

El modelado de superficie es un proceso que puede requerir conversiones entre diferentes tipos de modelado en 3D. El flujo de trabajo de modelado de superficies típico es: Creación de un modelo en el que se combinen objetos de malla, superficies y sólidos 3D. Los modelos de superficie se crean mediante muchas de las mismas herramientas que se utilizan para los modelos sólidos: barrido, solevación, extrusión y revolución. También se pueden crear superficies mediante la fusión, el parcheado, el desfase, el empalme o el alargamiento de otras superficies.

Pasos en el modelado 3D de superficies

  1. Hacer un sketch “libre”.

  1. Construir una superficie.

  1. Hacer un nuevo sketch en esa nueva superficie. El sketch nuevo que genero tiene una forma parecida al anterior.

  1. Hacer un loft entre los dos sketch que formé pero en la sección superficie primero. En este caso forma solamente una superficie.

  1. Este paso lo hacemos para entender la diferencia entre un solido y una superficie. Hacer un loft entre los dos sketch pero esta vez en la sección de solidos. Quedan figuras muy parecidas pero en este caso se forma un solido.

  1. Hacemos un nuevo sketch en un nuevo plano de la superficie. Hacemos otra “Spline”. Luego extruimos la figura creada para ambos lados. Por lo que tendremos una superficie que intersecta a otra superficie.

  1. Queremos lograr que la primera superficie pase por la circunferencia creada en el paso anterior, que sea parte de esta. Para esto vamos al entorno de solido y aplicamos “dividir cuerpo”. Ahora tenemos 3 superficies y faltaria la del cuerpo que divide a la superficie inicial.

  1. Para obtener esa superficie, volvemos al entorno de Superficie y hacemos un trim.

  • Esto nos demuestra que siempre que yo construya 2 superficies podemos generar la intersección exacta entre ellas.
  1. Luego utilizamos una herramienta que se llama parche para generar una base para la superficie. Ahora tenemos una variedad de superficies, y debemos lograr llegar a tener un solido.

  1. Entonces para generar el solido, debemos usar la herramienta “coser”. Así hicimos que todas las superficies sean un solido.

  1. Luego le hacemos un fillet. Luego hacemos un shell, para generar un vacio.

Esto es muy util para hacer envases o contenedores, ya que partimos desde una superficie mas especifica; un receptor o un embase de agua dentro de un motor. De este modo yo puedo crear esa forma mas especifica. Por eso partimos desde una superficie.

  1. Como partimos a partir de superficies, yo puedo alterar estas para cambiar mi forma si es que deseo.
