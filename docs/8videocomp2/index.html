Cura; software de Impresión 3D.

Para poder realizar la impresión 3D se requiere de un software de corte, que muchas veces también es llamado laminador, y este se usa para poder convertir el modelo 3D en un archivo código G que proporcionará a la impresora 3D todas las instrucciones necesarias. Uno de estos programas es CURA.

Cura, fue desarrollado por David Braam en 2014 y luego adoptada por la empresa Ultimaker. Este es un software de código abierto, es el más extendido en el mercado mundial para la fabricación aditiva. Al ser código abierto, Cura está en constante evolución.

Una de las mejores características que tiene Cura, y que es por la cual es tan popular, es por su facilidad de uso. Además tiene soporte para diferentes formatos de archivo y compatibilidad con muchas impresores 3D.

Software- Pasos para Imprimir en Cura

  1. Seleccionamos la impresora que nosotros queramos, en este caso seleccionamos: Creality Ender-3. Además seleccionas “eSUN Purple PLA+” con 0.4 mm Nozzle.

  1. Abrimos un archivo STL que tengamos modelo. En mi caso abrí un anillo que modele para el ramo de Modelación 3D el trimestre pasado.

  1. Corroboramos las medidas en escala. Rotamos si es necesario.

  2. Cambiamos la configuración para imprimir. Oprimimos “Slice”. Este proceso se llama slice, ya que genera una serie de capas al objeto

  1. Así es como queda estructurado el anillo para imprimirse. (Perímetros interiores y exteriores)
